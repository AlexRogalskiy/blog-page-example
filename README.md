## Project setup
### Clone Repo
```
git@github.com:merakiui/blog.git
```
### Open Folder
```
cd blog
```

### Install NPM Modules
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
